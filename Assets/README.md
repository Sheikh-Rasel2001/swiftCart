1) What is the difference between null and undefined?
Ans: null = এখানে কোনো ভ্যালু রাখা হয়নি undefined = একটা variable declare করা হয়েছে কিন্তু কোনো ভ্যালু রাখা হয়নি 

2) What is the use of the map() function in JavaScript? How is it different from forEach()?
Ans: map() = object or array loop করে এবং নতুন একটা array return করে । 
 forEach() = object or array loop করে কিন্তু নতুন কোনো array return করে না । 

3) What is the difference between == and ===?
Ans: (==) শুধু ভ্যালু এর মধ্যে compare করে number থাকলে number থাকবে ।
(===) ভ্যালু এবং টাইপের এর মধ্যে compare করে number থাকলে string এ পরিবর্তন করে ।
